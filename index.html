OC Creator Toolkit (Mockup)

📅 Timeline Manager

  • Create and manage timelines per OC
  • Visual editor (drag events)
  • Future feature: Link multiple timelines

📏 Height Comparison Tool

  • Upload silhouette images
  • Use inch/foot markers for realism

🧬 Family Tree Builder

  • Create bloodlines, add notes to relationships
  • Supports complex/looped inbreeding

⚖️ Crime Calculator

  • Input OC's crimes
  • See estimated prison sentence and capital eligibility

🧭 Alignment Chart Maker

  • Lawful/Chaotic vs Good/Evil grid
  • Drag-and-drop OCs

🧠 Personality Quiz Maker

  • Create custom quizzes for characters/world
  • Share results or embed on site

📄 OC Profile Builder

  • Custom fields like name, species, backstory
  • Optional gallery, personality stats

🔗 Relationship Map

  • Connect OCs with lines (ally, enemy, relative)
  • Color-coded relationship types
